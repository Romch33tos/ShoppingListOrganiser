# 🛒 Организатор списка покупок

Многопользовательское веб-приложение для совместного ведения списков покупок с гибкой системой прав доступа.

---

## 📌 Описание

Приложение позволяет пользователям:
- создавать списки покупок;
- добавлять, редактировать и удалять товары;
- отмечать товары как купленные;
- оставлять комментарии к товарам;
- приглашать других пользователей по email с разными уровнями доступа.

## 🧱 Технологический стек

- **Backend:** Node.js + Express  
- **База данных:** MongoDB (документная модель)  
- **Шаблонизатор:** EJS  
- **Сессии:** express-session  
- **Язык:** JavaScript  

## 🗂️ Структура базы данных

### Коллекции:
- `users` — пользователи (name, email, passwordHash, registeredAt)  
- `shoppingLists` — списки покупок с встроенными товарами и комментариями  
- `permissions` — права доступа (listId, userId, role)
