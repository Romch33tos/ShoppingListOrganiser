# Организатор списка покупок

Организатор списка покупок – это многопользовательское веб-приложение, позволяющее создавать и совместно управлять списками товаров. Система поддерживает разграничение прав доступа на уровне отдельных списков, что делает её идеальным инструментом для семей, рабочих групп и любых коллективов, планирующих совместные закупки.

## Ключевые возможности

* Регистрация и аутентификация пользователей
* Создание и управление списками покупок
* Совместный доступ с ролевой моделью:
  * Владелец – полный контроль над списком
  * Редактор – добавление, изменение и удаление товаров
  * Комментатор – оставление комментариев к товарам
  * Наблюдатель – только просмотр
* Управление товарами с указанием количества, единиц измерения и категорий
* Комментирование товаров для уточнения деталей
* Отметка о покупке товаров
* Статистика количества некупленных позиций

## Технологии

| Компонент | Технология |
| :--- | :--- |
| Серверная часть | Node.js + Express.js |
| База данных | MongoDB (документная модель) |
| Шаблонизатор | EJS |
| Управление сессиями | express-session |
| Стилизация | CSS (Flexbox, Grid) |

## Установка и запуск

### Требования

Перед началом убедитесь, что установлены:
* Node.js (версия 14 или выше)
* npm (менеджер пакетов)
* MongoDB Community Server (запущен на порту 27017)

### Шаг 1: Клонирование репозитория

```bash
git clone https://github.com/Romch33tos/ShoppingListOrganiser.git
cd shopping-list-organizer
```

### Шаг 2: Установка зависимостей

```bash
npm install
```

### Шаг 3: Настройка базы данных

Запустите скрипт инициализации для создания тестовых данных:

```bash
node init_db.js
```

### Шаг 4: Запуск сервера

```bash
node server.js
```

После запуска сервер будет доступен по адресу: [http://localhost:3000](http://localhost:3000)
